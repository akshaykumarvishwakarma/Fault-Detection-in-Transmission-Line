# Fault-Detection-in-Transmission-Line
A hardware-based transmission line fault detection system for identifying short circuit, open circuit, overload, and line faults using electrical and electronic components.
Fault Detection in Transmission Line
Project Description

The Fault Detection in Transmission Line project is designed to detect and indicate faults occurring in an electrical transmission line using a simple hardware-based prototype. The system continuously monitors the transmission line and identifies abnormal conditions such as short circuits, open circuits, and overload faults. When a fault occurs, the corresponding indication is displayed, allowing quick identification of the faulty section. This project demonstrates the basic principle of transmission line protection and fault monitoring using electrical and electronic components. It is developed as a low-cost educational model to improve understanding of power system protection and can be further enhanced with IoT, GSM, or SCADA for remote fault monitoring.

Working Principle

The system continuously monitors the transmission line conditions. Different switches are used to simulate transmission line faults such as short circuits, open circuits, and overloads. The Arduino processes the input signals and activates the corresponding LED indicators to identify the type of fault. This enables quick fault identification and demonstrates the working principle of transmission line protection systems.
